# House Price Prediction using Machine Learning

## 📌 Project Overview
This project aims to predict house prices based on various property features such as area, number of bedrooms, bathrooms, parking availability, and other amenities. The objective is to help buyers, sellers, and real estate businesses estimate property values using data-driven insights.

## 🎯 Objectives
- Perform data exploration and preprocessing.
- Handle missing values and duplicate records.
- Convert categorical features into numerical format.
- Train and evaluate machine learning regression models.
- Identify the most influential factors affecting house prices.
- Visualize key insights using charts.

## 📊 Dataset
**Dataset:** Housing Prices Dataset

Source: https://www.kaggle.com/datasets/yasserh/housing-prices-dataset

The dataset contains information about residential properties, including:
- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Air Conditioning
- Basement
- Preferred Area
- Furnishing Status
- House Price (Target Variable)

## 🛠️ Technologies Used
- Python 3.x
- Pandas

- Scikit-learn
- Jupyter Notebook / VS Code

## 🔄 Project Workflow
1. Data Loading & Exploration
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Train-Test Split (80:20)
5. Linear Regression Model
6. Random Forest Regressor Model
7. Model Evaluation
8. Data Visualization
9. Insights & Recommendations

## 📈 Evaluation Metrics
The models were evaluated using:
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## 📊 Visualizations
The project includes:
- House Price Distribution Histogram
- Correlation Heatmap
- Actual vs Predicted Price Scatter Plot
- Feature Importance Analysis

## 🔍 Key Findings
- Property area is one of the strongest factors influencing house prices.
- Bathrooms, parking spaces, and air conditioning significantly impact property value.
- Random Forest Regressor outperformed Linear Regression in prediction accuracy.
- Machine learning can effectively support real estate pricing decisions.

## 💡 Business Recommendation
Real estate companies should focus on property size, location quality, and essential amenities when determining pricing strategies. Predictive models can help improve pricing consistency and market analysis.
